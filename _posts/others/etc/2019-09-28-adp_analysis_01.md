```R
StatScore = c(88, 90, 87, 92, 75, 65, 93, 54, 85, 90)
```

**사용자 정의 함수**


```R
addto = function (a) {

  isum=0
  for (i in 1:a){
    isum = isum + i
  }
  print(isum)
}

print(addto(10))
print(addto(100))
```

    [1] 55
    [1] 55
    [1] 5050
    [1] 5050
